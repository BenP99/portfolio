Games

I create my games using the Unity engine which uses the language C#.

However I have also made a game using Java which allowed me to create a game from scratch, meaning I had to learn how to create a game loop and develop how the game is played and run within that loop. I did this using a tutorial online which helped me learn more about developing games and the back end functionality that helps keeps games running which are pre-developed for you in game engines like Unity.

A game I made with the Unity engine for my Final Project during my 3rd year at the University of Hertfordshire. This project was to make a Roguelike game that offered solutions to three big problems thats persisted in a lot of Roguelike games that exist which include: Not having a Save Feature, Poor Difficulty Balancing, and Little to No Clear End Goals. To do this I planned the project in the Project Planning module the first semester along with doing some experimentation on the implementation of all the planned mechanics, and in the second semester I developed the project along with a project report that I handed in for the assignment.
